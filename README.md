## Overview

**DontLookAway** is a minimalist and modern web-based memory game focused on rhythm, recall, and precision.  
Each round, the game shows a sequence of directional arrows that the player must reproduce perfectly.

---

## Gameplay

- A sequence of directions is generated for the current run
- The game shows the first direction, then the first two, then the first three, and so on
- You must repeat the full sequence in the correct order
- A single mistake ends the run immediately
- The goal is to survive as long as possible and reach **100**

---

## Game Modes

| Mode    | Description |
|---------|-------------|
| Daily   | Same 100-direction challenge for everyone on that day |
| Endless | Fresh randomly generated run for unlimited practice |

The **Daily** sequence resets every day at **midnight, Paris time**.

---

## Controls

DontLookAway is designed to be simple and accessible on both desktop and mobile.

### Desktop
- Arrow keys
- WASD
- ZQSD
- Clickable on-screen arrows

### Mobile
- Swipe gestures
- Tap/click on the on-screen arrows

---

## Statistics

At the end of each run, DontLookAway displays performance information such as:

- Score reached
- Daily attempt number
- Correct inputs
- Average input speed
- Average input rate
- Accuracy
- Total input time

---

## Themes & Customization

DontLookAway includes visual customization options to fit different moods and preferences:

- Light mode
- Dark mode
- Multiple skins:
    - Ember
    - Ocean
    - Plum
    - Jade
    - Gold
    - Rose
    - Ice

The selected theme and skin are saved locally.

---


## Inspiration

DontLookAway takes inspiration from classic sequence-memory games such as **Simon**, while adopting a cleaner and more modern web-game approach with daily competition like Wordle.
